# OMAR AI: Advanced AI Desktop Agent

OMAR AI is a self-hosted AI desktop agent that automates computer tasks through natural language commands. It operates within a containerized Linux desktop, providing a secure and isolated environment for AI-driven automation.

## Features
- **Dark Mode Interface**: Modern and sleek dark UI with glow effects.
- **Palestine Support**: Proudly supporting the cause with integrated messages.
- **Autonomous Task Execution**: Describe your task in natural language and watch OMAR AI work.
- **Secure & Private**: Self-hosted on your own infrastructure.
